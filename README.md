# 🌾 Laxmi Flour Mill
### *Artisanal & Chemical Free • Fresh Chakki Atta, Milled Daily.*

Welcome to the digital home of **Laxmi Flour Mill**, where tradition meets modern excellence. This project is a premium-luxury web experience designed to showcase our artisanal stone-ground flour and curated kitchen essentials.

---

## ✨ Brand Philosophy
We believe in the purity of grain and the integrity of process. Our website reflects this through:
- **Artisanal Quality**: Every visual element is crafted to convey the luxury of small-batch milling.
- **Traditional Roots**: Celebrating the heirloom techniques of stone-cold milling.
- **Modern Performance**: A seamless, high-performance experience built on the latest web technologies.

## 🚀 Key Features
- **Modern Luxury UI**: A bespoke design system featuring glassmorphism, rich typography, and a warm, organic color palette.
- **Fluid CSS Animations**: Hand-crafted, lightweight animations that enhance user engagement without the weight of external libraries.
- **Custom Reveal System**: High-performance "in-view" transitions powered by the Intersection Observer API.
- **Fully Responsive**: Optimized for every device, from mobile handsets to high-resolution displays.
- **Direct Ordering**: Seamlessly integrated WhatsApp and Call CTAs for direct customer interaction.

## 🛠️ Tech Stack
- **Package Manager**: [pnpm](https://pnpm.io/)
- **Framework**: [Next.js 15+](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com) & Vanilla CSS
- **Icons**: [Lucide React](https://lucide.dev)
- **State Management**: React Hooks (useState, useEffect, custom useReveal)
- **Optimization**: Next/Font for premium typography (Playfair Display & Inter)

## 🛠️ Development Setup

First, install the dependencies:
```bash
pnpm install
```

Then, run the development server:
```bash
pnpm dev
```

Build for production:
```bash
pnpm build
```

Lint the project:
```bash
pnpm lint
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
